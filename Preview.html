üì• Installation Instructions

  1. Install a userscript manager extension like Tampermonkey (Chrome/Firefox) or Violentmonkey
  2. Install this script from Greasy Fork or directly from the source
  3. Navigate to Instagram in your browser and log in
  4. The finder tool button will appear in the bottom right corner of the screen

üìù Usage Instructions

  1. Click the eye icon in the bottom right corner of Instagram
  2. Adjust search limit and delay settings as needed
  3. Add any accounts you want to exclude to the whitelist
  4. Click "Find Non-Followers" to start the scan
  5. Watch the progress and results in real-time
  6. Filter results using the search box or filter dropdown
  7. Click "Unfollow" next to any user to unfollow them immediately
  8. Use "Mass Unfollow" to unfollow multiple users at once
  9. Use "Export List" to save results as a text file
  10. Use "Load Saved" to reload previous scan results
  11. Click "Close" to hide the panel when finished

üîê Privacy & Security

  • This script only accesses Instagram.com through your logged-in session
  • Uses the same APIs that Instagram's website uses
  • No data collection or external transmission
  • All processing happens locally in your browser
  • Your login credentials are never stored or transmitted

üìå Important Notes

  • Warning Using too small of a delay between requests may trigger Instagram's rate limiting
  • Recommended Default delay: 2000-3000ms (2-3 seconds) works well for most accounts
